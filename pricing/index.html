Compliance statement

"Paid access never influences public Sigñal scores, rankings, or review outcomes."

Sigñal scores are the output of editorial review of publicly available brand marketing claims and visible evidence. Reviewers do not have access to commercial account information when scoring. Brand Pro subscribers receive workflow tools and analytical access to their own data — not editorial access, queue priority, or score modification capabilities. This separation is enforced structurally, not by policy alone.

Architecture — future database fields

Reference for implementation. Not user-facing.

users (additions)

tier                TEXT DEFAULT 'free'
                    -- 'free' | 'pro_consumer'
stripe_customer_id  TEXT UNIQUE
subscription_status TEXT
                    -- 'active'|'canceled'|'past_due'
subscription_ends_at TIMESTAMPTZ

brand_accounts (new table)

id               UUID PRIMARY KEY
domain           TEXT UNIQUE NOT NULL
email            TEXT NOT NULL
tier             TEXT DEFAULT 'free'
                 -- 'free' | 'brand_pro'
stripe_customer_id TEXT
claimed_at       TIMESTAMPTZ
verified_at      TIMESTAMPTZ

watchlists (new — pro_consumer)

id                  UUID PRIMARY KEY
user_id             UUID REFERENCES users
domain              TEXT NOT NULL
alert_score_change  BOOLEAN DEFAULT true
alert_claim_change  BOOLEAN DEFAULT true
created_at          TIMESTAMPTZ

api_keys (new — data tier)

id                UUID PRIMARY KEY
key               TEXT UNIQUE NOT NULL
customer_name     TEXT
tier              TEXT
                  -- 'starter'|'growth'|'enterprise'
rate_limit_per_hr INTEGER DEFAULT 100
active            BOOLEAN DEFAULT true
created_at        TIMESTAMPTZ

brand_audits (new — brand_pro)

id               UUID PRIMARY KEY
brand_account_id UUID REFERENCES brand_accounts
submitted_claims JSONB
status           TEXT DEFAULT 'pending'
reviewer_notes   TEXT
submitted_at     TIMESTAMPTZ
reviewed_at      TIMESTAMPTZ
-- Never auto-updates public score.
-- Advisory output only.
